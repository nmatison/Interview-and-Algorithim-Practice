# Dynamic Programming

## Tabulation vs Memoizatation
Tabulation:
+ Bottom Up (bottom case to the top case)
+ dp[0] --> dp[n]

Memoization:
+ Top Down (top case to the bottom case)
+ dp[n] --> dp[0]
